# Protocol-Buffers-Python
Google Protocol BuffersのPythonでのサンプルです

## 初回起動の前に(インストール済みならばこの手順をスキップしてください)

        pip install google
        pip install protobuf
        
## 次に
環境に合わせてprotoc.shの$SRC_DIRをRootDirに、$DST_DIRをOutputDirにしてください

        chmod 777 ./protoc.sh
        ./protoc.sh
        
を実行してください
